## Overview

Configuration location: `~/.config/hypr/`

See the [README in your hypr folder](https://github.com/binarylinuxx/dots/tree/main/.config/hypr) for detailed documentation.

## Quick Reference

- Main config: `hyprland.conf`
- Keybindings: `keyboard/binds.conf`
- Visual settings: `visual/`
- Window rules: `rules/`
- Scripts: `scripts/`
