## Automated Publishing

Use the provided scripts in `arch-deps/`:

```bash
# Interactive mode
./publish-to-aur.sh

# Quick mode (no prompts)
./quick-publish.sh
```

See the [AUR Quick Start guide](https://github.com/binarylinuxx/dots/blob/main/arch-deps/AUR-QUICK-START.md) for details.
