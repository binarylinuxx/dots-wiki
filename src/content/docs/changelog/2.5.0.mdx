## Added

### Technical

- **blx-sshell** has been heavily tested and reworked to support portable devices (laptops)
- Implemented a simple, easy-to-use **WiFi menu** so you can connect to a network out of the box

### Desktop Widgets

- The **weather widget** can now use the [OpenWeatherMap](https://openweathermap.org/) API for more precise location than the automatic IP-based detection

### Desktop

- Added a **sidebar** with quick toggles

## Changed

- Improved **NetworkManager service** — now works reliably
- **Battery service** is now custom-built
