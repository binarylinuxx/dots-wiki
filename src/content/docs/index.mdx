<script>{`
	var showcaseIdx = 0;
	var showcaseTotal = 3;
	function showcaseSet(i) {
		showcaseIdx = i;
		document.getElementById('showcase-track').style.transform = 'translateX(-' + (i * 33.33) + '%)';
		document.querySelectorAll('.showcase-dot').forEach(function(d, j) {
			d.classList.toggle('active', j === i);
		});
	}
	function showcaseGo(dir) {
		var next = showcaseIdx + dir;
		if (next < 0) next = showcaseTotal - 1;
		if (next >= showcaseTotal) next = 0;
		showcaseSet(next);
	}
`}</script>

## What's Included

- **Hyprland** - Dynamic tiling Wayland compositor
- **Quickshell** - Custom QML-based shell and widgets
- **Matugen** - Material Design 3 color generation
- **Fish Shell** - Modern, user-friendly shell
- **Starship** - Fast, customizable prompt
- **Ghostty** - GPU-accelerated terminal emulator
